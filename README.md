# Dashboard de Indicadores Estratégicos - Excel

Este repositório contém o projeto prático desenvolvido para consolidar os conhecimentos adquiridos no curso **Recursos Visuais com Excel: explorando gráficos e formatos** da Alura. 

O objetivo do projeto foi aplicar técnicas avançadas de design, estruturação de dados e funções analíticas para transformar dados brutos em um painel visual dinâmico, focado na tomada de decisão estratégica.

## 📋 Visão Geral do Projeto

Utilizando uma base de dados e uma marca fictícias, o dashboard foi planejado para simular um cenário real de análise de negócios. O painel prioriza a experiência do usuário (UX) e o storytelling com dados, permitindo identificar metas, gargalos e tendências de forma imediata.

## 🛠️ Recursos Visuais e Técnicas Aplicadas

* **Estruturação de Dashboards:** Planejamento de layout e organização de indicadores-chave (KPIs).
* **Formatação Condicional Avançada:** Criação de alertas visuais automáticos para destacar pontos críticos.
* **Gráficos Dinâmicos:** Seleção e customização de gráficos específicos para cada contexto de negócio.
* **Gráficos de Relógio (Velocímetro):** Construção passo a passo de um monitor intuitivo de atingimento de metas.
* **Design e Refinamento:** Técnicas de alinhamento, segmentação de dados, limpeza visual e remoção de ruídos de interface.

## 🧮 Funções e Lógicas Utilizadas

A inteligência por trás do painel foi construída utilizando as seguintes funções do Excel:

* `PROCX`: Correção e busca cruzada de dados entre tabelas de produtos e registros de vendas.
* `SOMASE` / `SOMASES`: Consolidação dinâmica do faturamento por categoria e por vendedor.
* `CONT.SE`: Mensuração de volume e quantidade de transações.
* `MÉDIA`: Definição de parâmetros dinâmicos e limites para os ponteiros do velocímetro.

## 🖥️ Como Visualizar o Projeto

1. Baixe o arquivo `.xlsx` disponível neste repositório.
2. Abra o arquivo no Microsoft Excel (recomendado utilizar versões recentes para total compatibilidade com a função `PROCX` e recursos de segmentação).
